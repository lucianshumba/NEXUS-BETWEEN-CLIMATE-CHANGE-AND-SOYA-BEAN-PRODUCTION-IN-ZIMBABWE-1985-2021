# NEXUS-BETWEEN-CLIMATE-CHANGE-AND-SOYA-BEAN-PRODUCTION-IN-ZIMBABWE-1985-2021
This study investigates how climate change affects Zimbabwe's soya bean production (1985–2021). Using ARDL and ARIMA models, it finds increasing rainfall negatively impacts yields, while temperature has no significant effect. Production is forecast to decline ~24% by 2030.
